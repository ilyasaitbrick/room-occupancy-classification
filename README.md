# Room Occupancy Classification (MAP-MLF - ISEP)

## Project Description

This project focuses on classifying the number of people in a room using radar-based images and deep learning models.

## Methodology

* Convolutional Neural Networks (CNN)
* Residual blocks & attention mechanisms
* Stratified K-Fold Cross Validation
* Test Time Augmentation (TTA)
* Pseudo-labeling (semi-supervised learning)

## Results

* Validation accuracy: ~97%
* Kaggle score: ~0.97+

## Files

* `notebook.ipynb` → full training pipeline
* `submission.csv` → final predictions
* `submission_round1.csv` → intermediate predictions

## How to run

1. Open notebook in Google Colab
2. Enable GPU
3. Upload dataset to Google Drive
4. Run all cells
